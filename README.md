# Md Nayeem Lapasam Portfolio

## Preview locally
Open `index.html` in a browser or use the Live Server extension in Visual Studio Code.

## Project structure
- `index.html` — complete portfolio website
- `assets/images/` — all website image assets
- `vercel.json` — static Vercel deployment configuration

The hero layout uses a larger integrated portrait, a warm framing glow, and a subtle floor shadow.

Final hero typography: role 21px, introduction 18px with 1.8 line-height.
